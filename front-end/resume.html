Honeypot

2021 | Ubuntu/Kali

I began by setting up a honeypot on an Ubuntu VM, installing a software called Cowrie. To test its efficacy, I utilized another VM running Kali to conduct a scan for open ports using zenmap. Upon discovering an open port, I opted to employ hydra, a tool equipped with a list of common passwords, and executed the software. Despite observing numerous login attempts in the logs on the Ubuntu machine, none were denied. In response, I turned to another tool called fail2ban to prevent potential brute force attacks and configured it to notify me via email if more than three attempts failed. The primary goal of this project was to create an enticing environment for users to connect into.
